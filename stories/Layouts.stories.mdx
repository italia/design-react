# Le Griglie

Nella definizione del layout di una interfaccia utente, la griglia Ã¨ una suddivisione dello spazio tramite precise spaziature verticali ed orizzontali.

Il Design React Kit eredita la potente griglia flexbox mobile-first di Reactstrap per costruire layout di tutte le forme e dimensioni grazie a un sistema a dodici colonne.

## Come funziona

Il sistema di griglie di Reactstrap usa una serie di contenitori, righe e colonne per disporre ed allineare i contenuti, costruito su flexbox e completamente responsive.
