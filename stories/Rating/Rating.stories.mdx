### Con label

Per visualizzare una label utilizzare l'attributo `legend` del componente `Rating`. In questo caso viene passato un nodo JSX all'attributo, che però accetta anche una struttura JSON (vedi documentazione).

            La label con il numero di stelle dev’essere contenuta in uno <code>&lt;span&gt;</code> semplice.</p>

### Sola lettura

Aggiungendo l'attributo `readOnly` si ottiene un Rating non modificabile di sola lettura.

<div className="docs">
    <Callout color="success" >
        <CalloutText>
            <p>Per rendere accessibile il contenuto è necessario aggiungere l'attributo <code>&lt;legend&gt;</code> con classe <code>.sr-only</code>.</p>
        </CalloutText>
    </Callout>
</div>

<Preview>
    <Story id="componenti-rating--rating-read-only" />
</Preview>

I seguenti valori per l'attributo `legend` sono equivalenti:

```jsx
// JSON
{ content: 'Valutazione 4 stelle su 5', srOnly: true }
// JSX
<legend className="sr-only">Valutazione 4 stelle su 5</legend>
```
